# selitys

## pdf rakennus
lataa seuraavat paketit käyttämästäsi paketin hallintajärjestelmästä:
``` sh
pacman -S texlive  
pacman -S inkscape
pacman -S evince
```
  
Sitten suorita:
``` sh
build.sh -pv

build.sh -op

``` 

Jos pdf ei rakennu, kokeile itse ratkoa ongelma.
